# BotBoard-Maintenance-API
A RESTful backend for managing routines, tasks, and devices in the BotBoard ecosystem. It enables the mobile app to perform CRUD operations and interact with maintenance data efficiently, ensuring seamless integration and communication between the app and backend services.


## Environment Variables

To run this project, you will need to add the following environment variables to your `.env` file:

- `DB_HOST`: The hostname of your database.
- `DB_USER`: The username for your database.
- `DB_PASS`: The password for your database.
- `DB_NAME`: The name of your database.
